target 속성 : 연결된 페이지를 어느 탭에서 열 것인지 지정

1) _self (기본값) : 현재 페이지(탭)에서 새로고침 되면서 열린다.
2) _blanks : 새 탭에서 열린다
3) _parent : 부모창의 탭에서 열린다
4) _top : 현재 실행중인 윈도우의 전체 상위 탭에서 열린다.
